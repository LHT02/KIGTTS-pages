训练端许可证与使用协议

本页面适用于 KIGTTS Electron 训练端和配套 Python 后端。训练端用于整理语料、蒸馏、Piper 训练与导出 Android 端可导入的 .kigvpk 语音包。

开源许可证

KIGTTS 训练端随项目源码以 GPL-3.0 发布。你可以在遵守 GPL-3.0 条款的前提下使用、复制、修改和分发训练端代码及其修改版本。

  • 源码仓库：LHT02/KIGTTS
  • 训练器下载：ModelScope / Hugging Face
  • 分发修改版时，应保留许可证文本、版权声明与第三方组件声明。

主要第三方组件

训练端涉及 Electron、React、Python、PyTorch、Piper、piper-phonemize、eSpeak NG、sherpa-onnx、ONNX Runtime 和 Python 依赖树等组件。

  • Piper vendored snapshot：仓库声明本地许可证为 MIT，活跃上游为 OHF-Voice/piper1-gpl。
  • eSpeak NG：上游许可证族为 GPL v3+，用于语音包相关发音数据。
  • ONNX Runtime：MIT；Python 依赖树各自遵循对应 dist-info 或 LICENSE 文件。

训练素材与语音包

训练端处理的录音、文本、参考音频、蒸馏语料、模型权重和导出的 voicepack.kigvpk 均由用户自行确认来源、授权范围和可分发范围。

  • 请勿使用未获授权的人声、角色语音、商业音色或受限制数据集训练并公开分发。
  • 使用 GPT-SoVITS、VoxCPM2 或其他外部模型蒸馏时，应额外遵守对应模型与数据许可证。
  • 导出的语音包不自动继承 KIGTTS 项目授权；其授权取决于用户素材和训练链路。
